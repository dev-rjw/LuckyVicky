# LuckyVicky
[내일배움캠프_7조]팀 소개 웹페이지 : 뭐야? (What?)

## 소개
### "날 소개하자면 네잎🍀만 아프지"
안녕하세요! 저희는 7조 <b>럭7ㅣ비7ㅣ</b>조 입니다.<br>
긍정적인 마인드로 도파민 뿜뿜하는 저희 조 웹의 매력을 느껴보세요!<br><br>

저희 조의 <b>뭐야?</b> 는 '뭐'인 5명이 모여서 소개를 하기 위한 제목으로,<br>
궁금증을 유발하도록 지어보았습니다.<br><br>

저희 조의 디자인 컨셉은 네잎클로버🍀 입니다.<br>
그에 맞게 웹 디자인도 네잎클로버가 뿜뿜한 페이지로 만들어 보았습니다.<br><br>

1️⃣첫 페이지는 럭7ㅣ비7ㅣ조를 소개하는 페이지로,<br>
각 조원들의 리스트로 구성되어 있습니다.<br>
조원의 사진을 클릭하면 해당 조원의 상세페이지로 넘어갑니다.<br><br>

2️⃣상세 페이지에 들어오면 firebase로 각각의 데이터를 화면에 표현하였습니다.<br>
좌측은 팀원의 사진을 캐러셀로 구현했으며,<br>
우측에는 팀원 설명, MBTI, 장점, 스타일 및 협업 스타일 / 블로그, 깃허브의 링크 / 코멘트와 좋아요 / 팀원이 사는 지역의 날씨까지 구현하였습니다.<br><br>

## 역할
- 김민규
    - 팀원 상세페이지 구현
        - 사진 캐러셀 구현 - firebase 사용
            1. 슬라이드 자동 롤링 효과 적용
            2. 썸네일 영역(작은 이미지)를 클릭하면 해당 index에 맞게 롤링되는 영역(큰 이미지)이 슬라이드 되어야한다.
            3. 이전, 다음 버튼 클릭 시 해당 슬라이드가 이동되어야 하며 썸네일 또한 그에 맞게 엑티브가 되어야한다.
            4. 슬라이드가 끝에 도달했을 때 loop 되어야 한다.
            5. 드래그하여 슬라이드가 이동되어야 한다.
- 류지원
    - 팀원 상세페이지 구현
        - 탭 구현
          - 탭 별로 상세 내용 변경되도록 구현 
        - 상세 내용 표현(팀원 설명, MBTI, 장점, 스타일 및 협업 스타일, 사진, 블로그, 깃허브) - firebase 사용
- 박준호
    - 메인 인트로 구현
      - 메인 좌측 팀 소개 구현(팀 설명, 팀 특징, 팀 목표, 팀 약속)
        1. 메인 페이지 방문을 기록 하게되는 Total 기능 추가
        2. 스크롤 시 좌측에 sticky되어 따라다니도록 구현
      - 메인 우측 팀원 프로필 구현(사진, 블로그, 깃허브) - firebase 사용
        1. 파이어베이스를 이용하여 팀원의 프로필을 출력
        2. 각 팀원의 블로그 주소에 맞춰 아이콘 변경되게 구현
        3. 스크롤에 따라 팀원 프로필이 순차적으로 보여지도록 구현
    - 메인 style 작업
- 이창현
    - 팀원 상세페이지 구현
        - 댓글 달기 구현 - firebase 사용
           1. 댓글 파이어베이스에 저장
           2. 댓글 삭제 기능
           3. 댓글 수정 기능 제작중
- 조아영
    - 팀원 상세페이지 구현
      - 팀원이 사는 지역의 날씨 추가
        1. Open Weather Map API를 이용
        2. firebase로 각 팀원의 사는 지역을 받고 해당 지역의 날씨가 보여지게 구현
      - 페이지 좋아요 구현 - firebase 사용
        1. 클릭 시 좋아요 수가 증가
        2. 증가한 좋아요 수가 실시간으로 반영
      - 메인, 상세 스타일 작업
        1. 사이트 스타일 통합 작업 진행
        2. 서브페이지 css 작업 진행
